## Classifier on famous UC Merced LU dataset

This is a mini practise project wherein I chose the famous [UC Merced](http://weegee.vision.ucmerced.edu/datasets/landuse.html) dataset and built a classifier on it using the [fastai](https://course.fast.ai/) library.

### References

+ Yi Yang and Shawn Newsam, "Bag-Of-Visual-Words and Spatial Extensions for Land-Use Classification," ACM SIGSPATIAL International Conference on Advances in Geographic Information Systems (ACM GIS), 2010.
